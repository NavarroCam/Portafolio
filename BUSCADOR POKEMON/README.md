# 🔍 Pokémon Finder - PokéAPI Project

Este proyecto es un buscador de Pokémon interactivo que utiliza la **PokéAPI** para obtener y visualizar datos en tiempo real. Está diseñado con un enfoque en la experiencia de usuario (UX) y una estética moderna inspirada en la franquicia Pokémon.

## 🚀 Características

* **Búsqueda en Tiempo Real:** Consulta cualquier Pokémon por su nombre.
* **Visualización Dual:** Muestra el Arte Oficial y la versión **Shiny** (variocolor) del Pokémon al mismo tiempo.
* **Datos Técnicos:** Extrae información detallada como tipo, peso y altura directamente de la API.
* **Estadísticas Base:** Incluye barras dinámicas para visualizar HP, Ataque y Defensa.
* **Diseño Estético:** Interfaz responsiva con fondo temático y contenedores con estilo *Glassmorphism*.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Flexbox, CSS Grid, animaciones `@keyframes` y gradientes dinámicos.
* **JavaScript (ES6+):** Uso de `fetch` y `async/await` para el consumo de la API de forma eficiente.
* **PokéAPI:** Fuente de datos externa para la información de los Pokémon.
